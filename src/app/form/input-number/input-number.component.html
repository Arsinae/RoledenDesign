Input Number
HTML Tag
rd-input-number
Parameters:
@Input() value: string Default value of the Input
@Input() label: string Text of the Label
@Input() color: string gold(default), green, red, blue, blue-light, purple, orange
@Input() display: string Type of display for the incrementor; 'single', 'both'(default)
@Input() min, @Input() max: number Min and Max value for the input; none by default
@Input() increment: number The value incremented for each click; 1(default
@Input() disabled: boolean Disable the input; false by default
@Output() valueChange: string Event emitted when the text has been changed
